# Section 7 QUICK REFERENCE: Disaster Protocols Cheat Sheet
## Emergency Response for Every Crisis Scenario

---

## MARKET CRASH PROTOCOLS

### Level 1: Market Down -5%
```
[ ] STOP all new entries immediately
[ ] ASSESS current positions (count, exposure, heat)
[ ] TIGHTEN all stops by 50% closer
[ ] CHECK broker platform functionality
[ ] PREPARE for worse (have phone number ready)
```

### Level 2: Market Down -10%
```
[ ] CLOSE 50% of all positions at market
[ ] PLACE hard stops at breakeven on remaining 50%
[ ] CANCEL all working orders
[ ] MOVE to 100% cash if:
    - Down >5% on day already
    - Market continues dropping
    - VIX above 40
[ ] DO NOT buy the dip or revenge trade
```

### Level 3: Market Down -15%+ or Circuit Breaker
```
[ ] CLOSE ALL positions 100% at market
[ ] STOP trading entirely (minimum 5 days)
[ ] ASSESS damage (before/after balance)
[ ] CONTACT risk partners (mentor, family)
[ ] WAIT for stabilization (3 green days + VIX <30)
```

---

## FLASH CRASH PROTOCOL

### Stock Drops 5-10% in <15 Minutes
```
STEP 1: Wait 2 minutes (let panic pass)
STEP 2: If stops hit → accept fill, move on
STEP 3: If stops not hit → assess real vs glitch
STEP 4: If real crash → exit at market within 5 min
STEP 5: If glitch → hold, wait for recovery
```

## OVERNIGHT GAP DOWN PROTOCOL

### Position Gaps -10%+ on Open
```
Is position -10% or more?
  YES → EXIT immediately, don't wait

Did you plan to hold through event?
  NO → EXIT now, you broke rules

Earnings gap or news gap?
  EARNINGS → EXIT 100%
  NEWS → Assess temporary vs permanent
  
[ ] Sell at market open
[ ] Calculate damage
[ ] Check if circuit breaker triggered
[ ] Journal what went wrong
[ ] NO revenge trading (24 hours off)
```

## PERSONAL DISASTER PROTOCOLS

### Medical Emergency / Cannot Trade
```
BEFORE (Preparation):
[ ] Emergency contact list created
[ ] Position list updated daily (cloud storage)
[ ] Family has broker phone number
[ ] Emergency instructions documented

WHEN IT HAPPENS:
[ ] Call broker within 1 hour
[ ] Say: "Medical emergency, close all positions"
[ ] Confirm: Account 100% cash
[ ] IF you can't call: Family calls broker
[ ] IF neither: Stops will eventually hit
```

### Extended Travel / Limited Internet
```
BEFORE TRAVEL:
[ ] Close all positions (move to 100% cash)
[ ] OR keep 1-2 positions with breakeven stops
[ ] Mobile trading app installed and tested
[ ] Broker phone number saved
[ ] Backup internet plan (hotspot, hotel wifi)

DURING TRAVEL:
[ ] Check positions once daily
[ ] Exit if market crashes
[ ] Call broker if no internet
```

## TECHNICAL DISASTER PROTOCOLS

### Internet Outage
```
[ ] Attempt reconnect (2 min): router, hotspot, wired
[ ] Call broker immediately if still down
[ ] Broker places orders by phone
[ ] Say: "Close all positions at market" (if crash)
[ ] Verify execution after internet restored

PREVENTION:
[ ] Broker phone number in phone contacts
[ ] Mobile hotspot as backup
[ ] Position list on paper daily
```

### Platform Crash
```
[ ] Determine scope: your PC or broker issue?
[ ] Try: Restart, clear cache, different browser
[ ] Use broker mobile app as backup
[ ] Use broker web backup platform
[ ] Call broker and trade by phone if needed
[ ] Close all positions if market volatile

PREVENTION:
[ ] Mobile app installed
[ ] Backup platform URL known
[ ] Test mobile app monthly
```

### Broker Failure/Bankruptcy
```
[ ] Check SIPC status (up to $500k insured)
[ ] Download all statements immediately
[ ] Screenshot positions and balances
[ ] Contact bankruptcy trustee
[ ] File SIPC claim
[ ] Wait 3-6 months for recovery
[ ] Open new broker account immediately

PREVENTION:
[ ] Use top-tier brokers only
[ ] Verify SIPC membership
[ ] Keep copies of statements monthly
```

## ACCOUNT DISASTER PROTOCOLS

### Large Drawdown (-15% to -25%)
```
[ ] STOP all trading (30 days minimum)
[ ] Close all positions (100% cash)
[ ] Withdraw portion to separate savings
[ ] Full system audit (Week 1-2)
[ ] External review with mentor (Week 2-3)
[ ] Rebuild plan (Week 3-4)
[ ] Paper trade validation (30 days)
[ ] Resume live with 50% size

AUDIT QUESTIONS:
- Did you break rules?
- Did system fail?
- Did environment shift?
- Did you over-trade?
```

### Catastrophic Loss (-25%+)
```
[ ] STOP trading immediately (90 days)
[ ] Withdraw all remaining capital
[ ] Psychological assessment (are you okay?)
[ ] Full forensic analysis (Month 1)
[ ] Professional help (Month 1-2)
[ ] Decision: Rebuild or stop trading?

DECISION POINT:
Option A: Rebuild (only if you understand what went wrong)
Option B: Stop trading (honest assessment)
```

### Margin Call
```
[ ] Add cash immediately if available
[ ] Close positions if no cash (losers first)
[ ] If do nothing → broker liquidates for you
[ ] After resolution → STOP USING MARGIN
[ ] Trade cash-only going forward

PREVENTION:
[ ] Trade cash-only (no margin)
[ ] If margin, keep <25% of account
[ ] Always use stops
```

## PSYCHOLOGICAL DISASTER PROTOCOLS

### Emotional Breakdown / Tilt

**Symptoms** (3+ = TILT):
- Revenge trading
- Over-trading (10+ per day)
- Ignoring stops
- Increasing sizes irrationally
- Can't sleep
- Obsessing over losses

**Actions**:
```
[ ] Close all positions NOW
[ ] Step away from computer
[ ] Physical intervention (walk 15-30 min)
[ ] Declare 3-day time out
[ ] Journal the breakdown
[ ] Talk to someone
[ ] Review trades objectively
[ ] Create prevention rule
[ ] Resume slowly (paper trade 5 days)
```

### Loss of Discipline / Rule-Breaking
```
[ ] Identify which rule you keep breaking
[ ] Understand WHY (too hard? don't believe it?)
[ ] Simplify rule OR strengthen consequence
[ ] Add accountability (trading partner checks)
[ ] Reduce trading frequency by 50%
[ ] Quality over quantity
```

## BLACK SWAN HEDGING (Optional - Advanced)

### VIX Call Options (Insurance)
```
Buy: VIX $30 calls, 90 days out
Cost: $1,000 per $50k account (2%)
Payoff: 300-1000% if VIX spikes to 40-50
Result: Offsets -20% portfolio loss

Pros: Cheap, huge crash payoff
Cons: Expires worthless 95% of time
```

### SPY Put Options (Direct Hedge)
```
Buy: SPY puts 5-7% out of money, 60 days
Cost: $3,000 per $50k account (6%)
Payoff: 200-500% if SPY drops -15%

Pros: Direct protection
Cons: Expensive, erodes if flat
```

### Inverse ETFs (Continuous)
```
Hold: 5-10% in SQQQ or SPXU
Cost: Decay over time
Payoff: +30% when market -10%

Pros: Always on, no expiration
Cons: Loses value in bull market
```

**Recommendation**:
- Start: No hedging (learn basics first)
- After 1 year: VIX calls (cheap insurance)
- After 2+ years: Consider puts or inverse ETFs
