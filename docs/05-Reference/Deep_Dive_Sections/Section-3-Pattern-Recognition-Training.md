# PATTERN #1: VOLATILITY CONTRACTION PATTERN (VCP)

## VISUAL REFERENCE EXAMPLES

[265]

**Key Visual Markers to Memorize**:
1. ✅ **Progressive tightening**: Each pullback smaller than prior (T-1 > T-2 > T-3)
2. ✅ **Volume contraction**: Volume bars shrinking during pullbacks
3. ✅ **Above 50 MA**: Price stays above rising 50-day moving average
4. ✅ **Tight final contraction**: Last pullback 3-8% depth, 5-15 days duration
5. ✅ **Breakout volume surge**: Volume 50-150%+ above average on breakout

---

## VCP IDENTIFICATION CHECKLIST

Use this checklist when scanning charts:

```
STOCK: ____________  DATE: ________

PRIOR UPTREND:
[ ] Stock up 30%+ before pattern starts
[ ] Uptrend duration: 4-12 weeks minimum
[ ] Moving averages aligned bullish (21>50>200)

CONTRACTION MEASUREMENTS:
Contraction 1 (first pullback):
  Peak: $___  Low: $___  Depth: ___%  Duration: ___ days

Contraction 2 (second pullback):
  Peak: $___  Low: $___  Depth: ___%  Duration: ___ days

Contraction 3 (third pullback):
  Peak: $___  Low: $___  Depth: ___%  Duration: ___ days

[ ] Each contraction smaller than prior (depth & duration)
[ ] At least 3 contractions visible
[ ] Final contraction depth 3-8%

VOLUME PATTERN:
Avg volume Contraction 1: ___ M shares
Avg volume Contraction 2: ___ M shares
Avg volume Contraction 3: ___ M shares
[ ] Each lighter than prior

PIVOT POINT:
Pivot (highest point in pattern): $___
[ ] Clear resistance level

MOVING AVERAGE CHECK:
[ ] Price never closed below 50 SMA during pattern
[ ] 21 EMA stayed above 50 SMA
[ ] Both MAs rising

VCP GRADE: A (excellent) / B (good) / C (acceptable) / F (reject)
```

---

## VCP PATTERN QUALITY GRADING

**Grade A (Trade with full size)**:
```
✓ 3-4 clean contractions, each 30-40% smaller than prior
✓ Volume contracts 40%+ each contraction
✓ Final contraction 3-5% depth, 5-10 days duration (very tight)
✓ Price stayed above 50 SMA entire pattern
✓ Pattern duration 8-12 weeks (ideal timeframe)
✓ Clear pivot point visible

Example: NVDA Oct 2023, SMCI Mar 2024
```

**Grade B (Trade with 70% of planned size)**:
```
✓ 3 contractions, each smaller but not perfectly progressive
✓ Volume contracts but not as dramatically
✓ Final contraction 5-8% depth, 10-15 days
✓ Price touched 50 SMA once but bounced
✓ Pattern duration 6-8 weeks or 12-16 weeks
✓ Pivot somewhat clear but not perfect

Example: CRWD May 2024
```

**Grade C (Trade with 50% size or skip)**:
```
⚠ Only 2 contractions (minimal)
⚠ Volume doesn't contract consistently
⚠ Final contraction 8-12% depth (wider than ideal)
⚠ Price broke below 50 SMA briefly during pattern
⚠ Pattern <6 weeks (rushed) or >20 weeks (too long)
⚠ Pivot ambiguous

Action: Only trade if 8+ edges present
```

**Grade F (Do not trade)**:
```
✗ Contractions erratic, not progressively smaller
✗ Volume spikes mid-pattern (distribution)
✗ Final contraction >15% depth
✗ Price broke significantly below 50 SMA
✗ Pattern >24 weeks (lost momentum)
✗ No clear pivot

Action: Remove from watchlist
```

# PATTERN #2: CUP-WITH-HANDLE

## VISUAL REFERENCE EXAMPLES

[267]

**Key Visual Markers to Memorize**:
1. ✅ **U-shaped cup**: Rounded bottom (not V-shaped), 20-60% depth
2. ✅ **Cup duration**: 5-12 weeks minimum (7-12 weeks ideal)
3. ✅ **Right side recovery**: Climbs back to within 5% of left-side high
4. ✅ **Handle formation**: 10-15% depth max, forms in upper half of cup
5. ✅ **Volume pattern**: Heavy on left side, light in handle, surge on breakout

---

## CUP-WITH-HANDLE IDENTIFICATION CHECKLIST

```
STOCK: ____________  DATE: ________

PRIOR UPTREND:
[ ] Stock up 30%+ before cup forms
[ ] Duration: 4+ weeks
[ ] Clean trend, not choppy

CUP MEASUREMENTS:
Cup left-side high: $___
Cup bottom (lowest low): $___
Cup depth: ___% = (High - Low) / High × 100
Cup duration: ___ weeks

[ ] Depth 20-60% (ideal 25-40%)
[ ] Duration 5-12 weeks
[ ] U-shaped bottom (2-3 weeks at low, not sharp V)

CUP RIGHT SIDE:
Right-side high (before handle): $___
Distance from left-side high: $___
Percentage: ___% = (Left High - Right High) / Left High × 100

[ ] Right side within 5-10% of left-side high
[ ] Right side took 2-4 weeks to form

HANDLE MEASUREMENTS:
Handle high: $___
Handle low: $___
Handle depth: ___% = (High - Low) / High × 100
Handle duration: ___ days

[ ] Depth 10-15% max (ideally 8-12%)
[ ] Duration 5-20 days (1-4 weeks)
[ ] Handle formed in upper 50% of cup
[ ] Handle doesn't undercut cup low

VOLUME PATTERN:
Volume during cup left side: High / Medium / Light
Volume during cup bottom: High / Medium / Light
Volume during handle: High / Medium / Light
[ ] Handle volume 20-50% below cup average

PIVOT POINT:
Pivot (right-side high or handle high): $___
[ ] Clear resistance at pivot

CUP-WITH-HANDLE GRADE: A / B / C / F
```

---

## CUP-WITH-HANDLE QUALITY GRADING

**Grade A (Trade with full size)**:
```
✓ Cup depth 25-35% (Goldilocks zone)
✓ Cup duration 7-12 weeks
✓ Perfect U-shape (spent 2-3 weeks at bottom)
✓ Right side within 2-5% of left-side high
✓ Handle depth 8-12%, duration 2-3 weeks
✓ Handle volume dried up 40%+ vs cup average
✓ Breakout volume 50%+ surge
✓ Price never broke 50 SMA during cup

Example: META Mar 2024, AAPL Jan 2023
```

**Grade B (Trade with 70% of planned size)**:
```
✓ Cup depth 20-25% or 35-50%
✓ Cup duration 5-7 weeks or 12-16 weeks
✓ U-shaped but slightly sharper bottom
✓ Right side within 5-10% of left-side high
✓ Handle depth 12-15%, duration 3-4 weeks
✓ Handle volume lighter but not dramatically
✓ Breakout volume 30-50% surge

Example: MSFT Feb 2024
```

**Grade C (Trade with 50% size or skip)**:
```
⚠ Cup depth <20% or >60%
⚠ Cup duration <5 weeks or >20 weeks
⚠ Slightly V-shaped bottom
⚠ Right side 10-15% below left-side high
⚠ Handle depth 15-20%, or duration <5 days
⚠ Volume pattern unclear
⚠ Breakout volume <30% surge

Action: Only trade if exceptional edges present
```

**Grade F (Do not trade)**:
```
✗ Cup depth >70% (too deep, shows weakness)
✗ V-shaped bottom (no accumulation)
✗ Right side never recovered (>20% below left-side high)
✗ Handle >20% depth or forms at cup bottom
✗ Volume spikes in handle (distribution)
✗ Breakout on weak volume

Action: Remove from watchlist
```

# PATTERN #3: FLAT BASE BREAKOUT

## VISUAL REFERENCE EXAMPLES

[266]
[269]

**Key Visual Markers to Memorize**:
1. ✅ **Prior advance**: 30-70% gain in 4-10 weeks (strong momentum)
2. ✅ **Tight range**: 8-15% depth (max), ideally 10-12%
3. ✅ **Duration**: 5-12 weeks minimum
4. ✅ **Horizontal consolidation**: Price oscillates between clear support/resistance
5. ✅ **Volume contraction**: Volume dries up 30-50% during base
6. ✅ **Above key MAs**: Price stays above 21/50 EMAs entire base

---

## FLAT BASE IDENTIFICATION CHECKLIST

```
STOCK: ____________  DATE: ________

PRIOR ADVANCE:
Start of advance: $___
Peak (before base): $___
Gain: ___% = (Peak - Start) / Start × 100
Duration: ___ weeks

[ ] Advance ≥30% (ideally 40-70%)
[ ] Duration 4-10 weeks
[ ] Clean, strong uptrend

BASE MEASUREMENTS:
Base high: $___
Base low: $___
Base depth: ___% = (High - Low) / High × 100
Base duration: ___ weeks

[ ] Depth ≤15% (ideally 8-12%)
[ ] Duration 5-12 weeks
[ ] Consolidation relatively flat (not ascending or descending sharply)

BASE STRUCTURE:
Number of resistance tests (highs near base high): ___
Number of support tests (lows near base low): ___
[ ] At least 2 resistance tests
[ ] At least 2 support tests
[ ] Price oscillates cleanly between levels

MOVING AVERAGE POSITIONING:
Price vs 21 EMA: Above / Below / On
Price vs 50 SMA: Above / Below / On
[ ] Price stayed above 21 EMA entire base (or touched once max)
[ ] Price never closed below 50 SMA

VOLUME BEHAVIOR:
Avg volume during advance: ___ M
Avg volume in base: ___ M
Volume contraction: ___% = (Advance - Base) / Advance × 100

[ ] Base volume contracted 30%+ vs advance

PIVOT POINT:
Pivot (base high): $___
[ ] Clear, tested resistance

FLAT BASE GRADE: A / B / C / F
```

---

## FLAT BASE QUALITY GRADING

**Grade A (Trade with full size)**:
```
✓ Prior advance 40-70% over 6-10 weeks
✓ Base depth 10-12% (extremely tight)
✓ Base duration 5-7 weeks (sweet spot)
✓ 2-3 clean tests of resistance
✓ Volume dried up 40-50% in base
✓ Price never broke 21 EMA during base
✓ Breakout on 50-100%+ volume surge

Example: CRWD Jun 2024, MSFT Dec 2023
```

**Grade B (Trade with 70% of planned size)**:
```
✓ Prior advance 30-40% over 4-6 weeks
✓ Base depth 12-15%
✓ Base duration 5-7 weeks or 8-12 weeks
✓ 2 tests of resistance (minimum)
✓ Volume dried up 30-40%
✓ Price touched 21 EMA once during base but bounced
✓ Breakout on 30-50% volume surge

Example: NVDA Aug 2023
```

**Grade C (Trade with 50% size or skip)**:
```
⚠ Prior advance 25-30% (marginal momentum)
⚠ Base depth 15-18%
⚠ Base duration <5 weeks or >15 weeks
⚠ Only 1 resistance test (weak)
⚠ Volume didn't contract much
⚠ Price choppy around 21 EMA
⚠ Breakout on <30% volume surge

Action: Only if 8+ edges and excellent environment
```

**Grade F (Do not trade)**:
```
✗ Prior advance <25% (insufficient momentum)
✗ Base depth >20% (not flat, too deep)
✗ Base shows clear downtrend or breakdown
✗ Price broke significantly below 50 SMA
✗ Volume spiked mid-base (distribution)
✗ No clear pivot point
✗ Breakout on weak volume

Action: Remove from watchlist
```
